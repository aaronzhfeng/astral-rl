# NeurIPS Paper Checklist

This document contains the NeurIPS submission checklist responses for the ASTRAL paper.

## Checklist Items

### 1. Claims
- **Answer**: Yes
- **Justification**: Abstract and introduction clearly state contributions and limitations.

### 2. Limitations
- **Answer**: Yes
- **Justification**: Section 7.3 discusses limitations including single environment, baseline failure, and mode similarity.

### 3. Theory assumptions and proofs
- **Answer**: N/A
- **Justification**: This is primarily an empirical paper without theoretical claims.

### 4. Experimental result reproducibility
- **Answer**: Yes
- **Justification**: Full hyperparameters in Table 2, architecture details in Section 3, and all configurations in Appendix.

### 5. Open access to data and code
- **Answer**: Yes
- **Justification**: Code will be released upon publication. Environment uses standard Gymnasium CartPole.

### 6. Experimental setting/details
- **Answer**: Yes
- **Justification**: Tables 1 and 2 provide environment and training details.

### 7. Experiment statistical significance
- **Answer**: Yes
- **Justification**: Core experiments run with 3 seeds; standard deviations reported in Table 3.

### 8. Experiments compute resources
- **Answer**: Yes
- **Justification**: Appendix A.2 details computational requirements.

### 9. Code of ethics
- **Answer**: Yes
- **Justification**: Research conforms to NeurIPS ethics guidelines.
